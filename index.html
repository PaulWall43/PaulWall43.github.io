  • Triton Battles

    Competition that inspires academic exellence


    Triton Battles is a website where students can compete against themselves and each other using user-created index cards. The competition aspect of the website motivates students to learn in a different way and thus can aid in the retention of information and critical thinking under pressure.



    Triton Battles also supports a unique, programmable index card, where questions can be answered by code. The user code is then compiled and run by our application and review for correctness.
